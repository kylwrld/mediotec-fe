## Getting Started

### Run locally

1. Clone the repo
   ```sh
   git clone https://github.com/kylwrld/mediotec-fe.git
   ```
2. Enter directory
   ```sh
   cd mediotec-fe
   ```
4. Install NPM packages
   ```sh
   npm install
   ```
6. Run
   ```sh
   npm run dev
   ```
